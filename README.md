## Installation

1. Clone the repo:

   ```bash
   git clone https://github.com/Mia06-coder/alx-coffee-shop-app.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the app:

   ```bash
   expo start
   ```
